# Xvsh - Custom shell fully scriptable in Lua

## You know, we have a lot of things configured in Lua. AwesomeWM, Neovim, Wezterm, hell, why not make a Lua shell.
#
## And that is exactly what I made here.

# This is a linux shell, made in C, that is configurable and rice-able in Lua.
#
### RC File is at ~/.xvshrc
#
### In that file, put the path to your Lua scripts, and it will load them when the shell is launched.
#
### I provided an example main.lua script in xvsh-scripts/scripts/main.lua
#
#
